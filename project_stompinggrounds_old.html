Stomping Grounds

 Game where you get to control legs and feet of an Eagle, Sergal and Dragon in though 9 levels of furry-stomping goodness.

Developer: Torvid

Release: Mid-2021 (god willing)
Platforms: Windows, Linux




Questions and Answers

What is the game about?
You play as an Eagle, Sergal and Dragon from a top-down perspective, using abilities like stomp, throw, screech and fire-breath to kill invaders.
Controls?
The control-scheme is similar to Mobas like league of legends, but instead of a pvp arena it's a series of levels.
What will the game cost?
Free! (when it's done) If you want to play early builds of the game, I post those for $10 on my patreon, found here: https://www.patreon.com/Torvid
When is it coming out?
Probably some time around mid 2021. (god willing)
Where can I follow development, get updates?
The best place is the public telegram channel or discord server where I post screenshots and clips of my this project and others:

What platforms will it work on?
Windows and Linux (tested on linux mint with an nvidia graphics card)
How long have you been working on it?
Since late 2018
Will there be gryphons?
Yes, yes there will be. :>
How can I support you?
For now, https://www.patreon.com/Torvid Pretty much all the money I get here I put back into the project.
Modding?
There is no official support for mods, but if you want to decompile and mod the game, all power to you! If you want the source code / project files one the game is out, feel free to send me a message!
Support / Versions?
I will release the game and that's it. I'll only patch or update it if some critical, game-breaking problems are discovered. (like the game not working at all on some machines)
Your last game was made in Unreal, Why return back to Unity?
I got sick of the visual scripting working on the barn game. My initial prototypes for this game were actually in unreal! But I realized doing the dynamic mud would probably require modifying the engine itself, which sounded scary at the time.
